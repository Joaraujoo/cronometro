# ⏱️ Cronômetro

Um cronômetro funcional com botões para iniciar, pausar e resetar o tempo.

## ✨ Funcionalidades
- Contador com segundos e minutos
- Botão de start, pause e reset
- Exibição clara do tempo decorrido

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript
